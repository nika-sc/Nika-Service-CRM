# Middleware

Папка `app/middleware` содержит middleware‑слой приложения (вспомогательные компоненты, которые обрабатывают запросы/контекст до/после маршрутов).

## Что внутри

- **`auth.py`**: функции/обвязка для аутентификации/авторизации (используется совместно с Flask-Login и RBAC декораторами).
- **`request_logger.py`** (если присутствует): логирование запросов/ответов, фильтрация чувствительных данных.

## RBAC и доступ

Система прав реализована через:
- роли (`users.role`)
- таблицу `permissions`
- таблицу связей `role_permissions`

Проверка доступа выполняется декораторами:
- `@login_required`
- `@role_required(...)`
- `@permission_required(...)`
