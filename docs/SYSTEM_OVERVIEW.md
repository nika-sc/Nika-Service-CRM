# Краткий обзор системы Nika CRM

**Версия:** 2.3  
**Дата:** 2026-03-02

## Основные возможности

| Модуль | Функции |
|--------|---------|
| **Заявки** | Реестр (21 колонка), Канбан, Журнал; динамическая смена статуса; услуги, товары, оплаты; комментарии |
| **Клиенты** | Поиск, депозит, портал; управление паролем портала; история заявок |
| **Склад** | Товары, категории, поставщики; приход/расход; инвентаризация; логи операций |
| **Магазин** | Быстрые продажи, корзина; автозапись в кассу |
| **Финансы** | Касса, статьи; синхронизация с оплатами; отчёты |
| **Отчёты** | Дашборд, Продажи, Зарплата, Касса, Долги, Логи действий; период по умолчанию — 7 дней |
| **Настройки** | Пользователи, роли, статусы, услуги, типы устройств; шаблоны писем; SMTP |

## Интерфейс

- **Тема:** светлая / тёмная (переключатель в шапке)
- **Склад — список товаров:** новый дизайн таблицы под обе темы

## Права доступа (RBAC)

- `view_orders`, `edit_orders`, `view_warehouse`, `manage_warehouse`
- `view_finance`, `manage_finance`, `view_shop`, `manage_shop`
- `view_action_logs`, `manage_statuses`, `manage_users`, `manage_settings`

## Структура проекта

- `app/` — Flask-приложение (routes, services, models)
- `templates/`, `static/` — интерфейс
- `database/` — SQLite, миграции
- `docs/` — актуальная документация
- `save/` — архив (скрипты, тесты, отчёты, бэкапы БД)
