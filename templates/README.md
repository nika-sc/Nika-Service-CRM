# Templates (Шаблоны)

В папке `templates/` находятся HTML шаблоны Jinja2.

## Структура

- **`dashboard.html`**: главная страница (дашборд).
- **`settings.html`**: настройки (в т.ч. управление пользователями/ролями/правами).
- **`partials/`**: общие компоненты интерфейса (navbar/sidebar/фильтры периода).
- **`reports/`**: шаблоны отчетов.
- **`salary/`**: страницы зарплаты и детализация по сотрудникам.

## Периоды (preset/date_from/date_to)

Во многих страницах (дашборд/отчеты/магазин) используется единый фильтр периода:
- **preset** (например: `today`, `yesterday`, `last_7_days`, `current_month`, …)
- **date_from/date_to** (кастомный диапазон)

Если задан `date_from/date_to`, то preset игнорируется (кастомный диапазон приоритетнее).
